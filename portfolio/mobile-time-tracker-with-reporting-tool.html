.NET Android Geolocation GPS iOS Microsoft SQL Server Web API
Duration: 500 hours
Date: May 2014

Overview

Mobile app for woodworking company allowing to monitor the work of employees for each project. The system provides an ability to create feedback forms, define respondents and control feedback terms on account of different notifications. As soon as the feedback is gathered the system puts it in the storage, collects statistics and forms the final view.
